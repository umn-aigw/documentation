# Getting Started

This guide covers how to request access and complete initial setup.

## Eligibility

Access can be provisioned for:

- Individuals
- Research labs
- Departments and units
- Academic courses

## Requesting Access

Contact your local CESI requester to explore whether the AI Gateway is a good fit for your use case. The CESI requester will submit a request to begin account creation. You will need to provide an EFS string for billing, an initial budget cap, and details about who should be granted access.

If you are requesting access for a course, include any cross-listed course identifiers or alternative sections (for example, honors sections).

Ongoing management of teams is handled through the University's Grouper platform. As the requester for a group, you will be asked to attest to current membership once per year.

## Budget Recommendations

Start with a conservative budget cap and increase it after observing real usage.

## Network Access

The gateway is available on campus or when connected through UMN VPN. Visit [gateway.loonai.umn.edu/ui](https://gateway.loonai.umn.edu/ui) to get started

If this requirement limits your workflow, contact the service team to discuss options. Feedback on this policy is still being collected.

## First Login Checklist

1. Confirm your team assignment is correct.
2. Verify budget and rate limit settings.
3. Create at least one API key.
4. Run a short test request before scaling up.
