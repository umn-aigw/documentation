# LoonAI Gateway

The LoonAI Gateway provides University of Minnesota researchers, faculty, staff, and students with managed access to leading AI models through a unified API.

## What You Get

- OpenAI-compatible endpoint for multi-model access
- Team-based API keys and budget controls
- Chargeback billing with usage transparency
- Fast onboarding for individuals, labs, and classes

## Platform Overview

LoonAI Gateway is powered by [LiteLLM](https://www.litellm.ai), which routes requests to multiple model providers behind a consistent API layer.

This design gives teams one integration pattern while preserving flexibility to choose models based on price, speed, and capabilities.
