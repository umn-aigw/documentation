# Getting Started

This guide covers how to request access and complete initial setup.

## Eligibility

Access can be provisioned for:

- Individuals
- Research labs
- Departments and units
- Academic courses

## Requesting Access

Contact your local CESI requester to explore whether the AI Gateway is a good fit for your use case. The CESI requester will submit a request to begin the account creation. You'll need to provide an EFS string for billing, an initial budget cap, as well as details about who should be granted access. 

If you're requesting access for a course, remember to include any cross-listed course identifiers or alternative sections (honors sections for example). 

Ongoing management of teams is via the University's Grouper platform. As the requester for a group, you'll be asked to attest to the current membership once per year. 

## Budget Recommendations

Start with a conservative budget cap and increase after observing real usage. 

## Network Access

The gateway is available on campus or when connected through the UMN VPN.

If this requirement limits your workflow, contact the service team to discuss options. We're still collecting feedback on this decision. 

## First Login Checklist

1. Confirm your team assignment is correct.
2. Verify budget and rate limit settings.
3. Create at least one API key.
4. Run a short test request before scaling up.
